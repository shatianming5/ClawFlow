- AgentRuntime, Planner, Executor, Tool Registry, Memory, Trace, Checkpoint, Governance.
- CLI, FastAPI, Web Dashboard.
- Unified Example Applications and benchmark pipeline.

- Distributed task queue.
- Multi-model routing.
- Policy editor and approval workflow.
- Trace replay diff and failure recovery reports.

## 1.0 Developer Framework

- SDK templates.
- Plugin marketplace.
- Cloud deployment profiles.
- Evaluation leaderboard.
